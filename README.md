# Movie Review Summarizer – Advanced AI Final Project

## Project Overview
An end-to-end AI system that analyzes and summarizes movie reviews using:
- **Sentiment Classification** – Fine-tuned BERT (93%+ accuracy)
- **Review Summarization** – TF-IDF Summarization and Prompt Engineering
- **Word Embeddings** – Word2Vec (100-dim)

## AI Techniques Used (from course)
| Technique | Implementation |
|---|---|
| NLP – Text Preprocessing | Notebook 1 |
| Word Embeddings (Word2Vec) | Notebook 1 |
| Transformer / LLM (BERT) | Notebook 2 |
| Transfer Learning | Notebook 2 |
| Prompt Engineering | Notebook 3 |
| Few-shot Learning | Notebook 3 |
| TF-IDF Summarization | Notebook 3 |

## Dataset
- **IMDb 50K Movie Reviews** (Kaggle)
- Place `IMDB_Dataset.csv` inside the `data/` folder before running
